# Box ReApp

一个给 Box / BFR 用的 Android 客户端。
主要用来在手机上做配置管理、状态查看，还有一些日常操作。

---

## 🧭 这个 App 是干嘛的？

如果你已经在用：

* Box
* 或者 BFR（box_for_magisk）

那这个 App 可以当作一个**统一控制面板**来用。

你不需要再去手动翻配置、查版本、或者跑脚本，很多操作可以直接在手机里完成。

---

## ✨ 能做什么

目前主要覆盖这些东西：

### ⚙️ 基础管理

* 服务相关配置查看 / 调整
* 常用设置集中管理

### 📦 版本 & 更新

* 读取模块版本（`module.prop`）
* 检查模块更新（支持 box / bfr）
* 检查 App 更新并下载 APK

---

### 🌐 网络相关

* 订阅配置管理
* 网络相关设置入口整合
* 为 Box / BFR 场景做适配

---

### 📱 UI & 体验

* 基于 Jetpack Compose
* 动态主题 / 外观设置

---

## 🧩 关于风味（box / bfr）

这个 App 是双风味构建的：

* `box` → 给 Box 用
* `bfr` → 给 box_for_magisk 用

两者区别：

* 更新源不同
* 下载资源不同
* 但 UI 和功能基本一致

---

## ⚠️ 一些小问题

* GitHub API 有限流（更新检测偶尔会失败）
* box / bfr 发布时要注意资源命名匹配

---

## 📌 一句话总结

> 这个 App 不是代理工具本身
> 而是 **Box 生态的手机控制端**
